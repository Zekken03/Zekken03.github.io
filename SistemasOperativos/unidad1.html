Introducción a los Sistemas Operativos

LOS SISTEMAS OPERATIVOS son programas que actúan como una interfaz entre el hardware de la computadora y los usuarios. Se encargan de gestionar los recursos de la máquina, como la memoria, el procesador, los dispositivos de entrada/salida y el almacenamiento, para que todo funcione de manera eficiente y ordenada. Sin ellos, no podríamos interactuar fácilmente con las aplicaciones o el hardware.

Conceptos Clave:

> Kernel

Es el núcleo del sistema operativo. Controla todas las operaciones de bajo nivel y actúa como un puente entre el software y el hardware. El kernel se encarga de tareas como la administración de la memoria, la planificación de los procesos y el manejo de interrupciones.

Cuando enciendes tu computadora, el kernel se asegura de que el hardware (como la memoria o el procesador) y el software (programas) puedan comunicarse correctamente.
Si estás usando el navegador, escuchando música y descargando archivos, el sistema operativo organiza todo para que parezca que todo sucede al mismo tiempo, aunque el procesador va alternando entre las tareas.

> Multitarea

Es la capacidad de un sistema operativo para ejecutar múltiples tareas o procesos al mismo tiempo. En realidad, el procesador ejecuta una tarea a la vez, pero el SO gestiona el tiempo de CPU de cada tarea para que parezca que todas se ejecutan simultáneamente.

> Administración de Memoria

El SO controla cómo se utiliza la memoria RAM y asigna espacio a los programas en ejecución. También maneja la memoria virtual cuando la RAM física se queda corta.

Mientras juegas un videojuego y tienes otras apps abiertas, el sistema operativo decide cuánta memoria necesita cada programa, y si falta RAM, usa la memoria del disco duro para evitar que todo se congele
Cuando guardas un documento, el sistema operativo lo coloca en un espacio específico del disco duro y lo etiqueta para que puedas encontrarlo después, siguiendo las reglas del sistema de archivos (como NTFS o FAT32).

> Sistema de Archivos

Es la manera en que el sistema operativo organiza y almacena los archivos en discos duros u otros medios de almacenamiento. Cada SO puede usar diferentes tipos de sistemas de archivos, como NTFS, FAT32 o EXT4.

> Procesos

Un proceso es un programa en ejecución. El SO se encarga de la planificación de procesos, permitiendo que diferentes programas compartan los recursos del procesador de manera eficiente.

Al abrir un programa como Photoshop, el sistema operativo crea un "proceso" que asigna los recursos necesarios (memoria, CPU) para que funcione de manera independiente a otros programas abiertos.
Si conectas tu mouse o impresora, el sistema operativo usa un controlador (software especializado) para que tu computadora sepa cómo interactuar con ese dispositivo.

> Controladores de Dispositivos

Son programas que permiten que el sistema operativo se comunique con el hardware, como impresoras, tarjetas de red, cámaras, etc.

> Seguridad y Permisos

Los SO implementan mecanismos para proteger los recursos y datos del sistema, asegurando que los usuarios solo tengan acceso a lo que se les permite.

Cuando intentas instalar una nueva app, el sistema operativo te pide autorización antes de hacer cambios importantes para evitar que programas maliciosos lo hagan sin tu permiso.
